A modern file manager for WordPress with a React-based interface. Manage your WordPress files directly from the admin area with an intuitive UI.

== Description ==

RZ File Manager provides a modern, user-friendly interface built with React to manage your WordPress installation's files and folders directly within the admin dashboard.

**Features:**

*   Browse directories and files
*   Upload files
*   Download files
*   Delete files and folders
*   Create new folders
*   (Add more specific features here if applicable)

This plugin aims to simplify file management tasks without needing FTP/SFTP access.

== Installation ==

1.  Upload the `rz-file-manager` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Access the File Manager via the WordPress admin menu (Specify the exact menu location here, e.g., Tools -> RZ File Manager).

Alternatively:

1.  Download the plugin zip file.
2.  Navigate to 'Plugins > Add New' in your WordPress admin area.
3.  Click 'Upload Plugin', choose the downloaded zip file, and click 'Install Now'.
4.  Activate the plugin.
